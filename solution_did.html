DID PLUS소개

Introduction Of DID PLUS

DID PLUS는 누구나 운영할 수 있는 쉽고 간편한 비즈니스 최적화 솔루션입니다.
원하는 콘텐츠를 제작하여 스케줄 예약 및 업로드를 통해 디스플레이에 송출해줍니다.
Web과 Local 버전 중 고객사의 웹 환경에 알맞게 선택하여 사용하실 수 있습니다.

DID PLUS 솔루션 기능

Function Of DID PLUS

  • WEB 기반의 솔루션

    웹 브라우저 기반의 운영 솔루션으로
    높은 확장성과 쉬운 사용 환경을
    제공합니다.

  • 실시간 모니터링

    디스플레이 원격 모니터링이
    가능해 실시간으로 진행상황을
    볼 수 있습니다.

  • 쉽고 간편한 스케줄링

    원하는 날짜와 시간에 맞추어
    누구나 쉽고 간편하게
    스케줄링이 가능합니다.

DID PLUS 로컬 (Local)

Features Of DID PLUS Local

  • 1. WEB 기반의 솔루션 (단일 유저)

    웹 브라우저 기반의 사용 환경으로 쉽고 간편한 유저 인터페이스를
    제공하여 누구나 손쉽게 솔루션 운용이 가능합니다.

  • 2. 실시간 모니터링

    어디서든 디스플레이 원격 모니터링이 가능해
    실시간으로 진행상황을 볼 수 있습니다.

  • 3. 간편한 스케줄링

    원하는 날짜와 시간에 맞추어 손쉽게
    콘텐츠 스케줄링 및 배포가 가능합니다.

DID PLUS 웹 (Web)

Features Of DID PLUS Web

*로컬 버전의 기능은 포함되어 있습니다.

  • 1. WEB 기반의 솔루션 (복수 유저)

    웹 브라우저 기반으로 쉬운 사용 환경과, 복수 유저가 하나의
    서버에 접근하여 누구나 솔루션을 운용할 수 있습니다.

  • 2. 다양한 화면 구성

    운영자가 직접 화면 구성이 가능하여 다양한 콘텐츠를
    하나의 디스플레이에 분할 표출 구성이 가능합니다.

  • 3. 권한 부여 기능

    관리자가 여러 사용자 계정을 생성할 수 있고
    계정별 기기의 권한을 각각 부여할 수 있습니다.
