빌딩안내 시스템 소개

Building Information System

빌딩안내 시스템은 디스플레이와 운영시스템으로 구성되어
건물 내방객 또는 직원들에게 다양한 정보를 시각적으로 전달하는 시스템입니다.
각종 콘텐츠를 디스플레이 장치에 표출하여, 기업의 홍보, 다양한 생활정보,
안내사항 등 내방객에게 정보를 시각화하여 효과적으로 전달하는 시스템입니다.

빌딩안내 시스템 구성

Building Information System Configuration

빌딩안내 시스템 구성은 네트워크 기반으로 운영 PC에서 모든 시스템을 통합 원격·제어가 가능합니다.
타 시스템 서버와 데이터 연동(옵션)을 통한 화재 경보 알람·실시간 에너지 정보·발전현황 등 실시간 시스템 연동 표출이 가능합니다.

빌딩안내 주요기기

Building Information System Main Instrument

  • 빌딩안내 서버

    • 운영 및 DB서버
    • 스케줄 및 콘텐츠 보관
    • 표출 컨트롤러 관리
    • 윈도우 서버 OS 기반

  • 빌딩안내 운영PC

    • 표출 컨트롤러 원격 관리
    • 콘텐츠 표출 및 제어 (업로드)
    • 스케줄 편성 및 전송
    • 윈도우 OS 기반

  • 표출 컨트롤러

    • 콘텐츠 저장·재생
    • 스케줄 저장·재생
    • 자동 전원 On / Off
    • 윈도우 임베디드 OS 기반

  • 디스플레이 (LCD / DID)

    • 산업용 디스플레이
    • 다양한 화면크기 (32"-98")
    • 선명도를 위한 전용 패널 사용
    • 다양한 타입의 디스플레이
