통합 모니터링 시스템 소개

Intergrated Monitoring System

통합 모니터링 시스템은 종합방재실/중앙관제실에 구성되는 시스템으로
운영 시스템 PC화면을 멀티로 구성된 멀티비전에 통합 표출합니다.
종합적인 상황 관제 기능과 총괄적인 지휘 통제 환경을 제공합니다.

통합 모니터링 시스템 구성

Intergrated Monioring System Configuration

통합 모니터링 시스템 구성은 네트워크 기반으로 운영 PC에서 모든 시스템을 통합 원격·제어가 가능합니다.
이벤트 연동을 통한 자동 레이아웃 전환/표출이 가능하며 유사시 신속한 상황 전파 및 공유가 가능합니다.

통합 모니터링 주요기기

Intergrated Monitoring System Main Instrument

  • 비디오 월

    • 산업용 디스플레이
    • 24/7 운용
    • 초 슬림 화면 간격 (0.88~3.5mm)
    • 시리얼 통합 제어

  • 월 컨트롤러
    Wall Controller

    • 상황판 표출 구성 장비
    • IP 기반 : Wall Controller
    • H/W 기반 : Multi Wall Controller
    • 확장성과 효과적인 운용

  • 메인 컨트롤 시스템
    Main Control System

    • 통합 모니터링 총괄 제어/운영
    • 사용하기 쉬운 UI
    • 프리셋/전원제어 기능
    • 이벤트 시나리오 연동 가능

  • 인테리어 마감

    • 멀티비전 인테리어
    • 현장 상황을 고려한 디자인
    • 방염 마감재 사용
    • 근무자 친화적인 콘솔데스크
